# rlmN_frameshift

This repository contains work for "A frameshift mutation in the methyltransferase rlmN is associated with linezolid resistance in Mycobacterium tuberculosis".

## download_scripts

Contains all scripts needed to download the initial ATB bacterial assemblies used for the manuscript

## blast_search

Contains all scripts + data needed to re-run the BLAST search across all of the ATB bacterial assemblies

## mutation_analysis

Contains all downstream processing, including calculations to create Table 2.

## phylogenetic_analysis

Contains all processing for the phylogenetic work, including homology of _rlmN_ and production of Fig 1 (before some Illustrator color editing).

## reference_genomes

Contains all reference genomes used to complete the annotation search for _rlmN_
